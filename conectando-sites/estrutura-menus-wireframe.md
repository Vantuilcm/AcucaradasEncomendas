## Elementos Globais

### Header

- Logo Açucaradas Encomendas
- Menu de navegação principal
- Botão "Baixar o App"
- Botão de alternância de tema (claro/escuro)

### Footer

- Logo Açucaradas Encomendas
- Links rápidos
- Links legais
- Informações de contato
- Redes sociais
- Copyright

### Elementos Flutuantes

- Botão de WhatsApp para contato rápido
- Botão "Voltar ao topo" (aparece ao rolar a página)

## Responsividade

### Mobile (< 768px)

- Menu se transforma em menu hambúrguer
- Layout em coluna única
- Fontes redimensionadas
- Imagens otimizadas para tela menor

### Tablet (768px - 1024px)

- Layout adaptado com menos colunas
- Ajustes de espaçamento

### Desktop (> 1024px)

- Layout completo conforme wireframes acima
- Experiência visual expandida
