# Instruções para Gravação das Telas do Aplicativo

## Preparação do Dispositivo

### Requisitos de Hardware
- Smartphone: Samsung Galaxy S21 (conforme especificado no roteiro)
- Cabo USB de alta qualidade para conexão com o computador
- Computador com software de captura de tela instalado

### Configuração do Dispositivo
1. **Limpar a tela** do dispositivo para remover impressões digitais e manchas
2. **Desativar notificações**:
   - Acesse Configurações > Notificações
   - Ative o modo "Não perturbe"
3. **Configurar brilho da tela**:
   - Ajuste para 80% do brilho máximo
   - Desative o ajuste automático de brilho
4. **Verificar bateria**:
   - Conecte o dispositivo ao carregador durante a gravação
   - Certifique-se de que o dispositivo tenha pelo menos 50% de bateria
5. **Limpar memória**:
   - Feche todos os aplicativos em segundo plano
   - Reinicie o dispositivo antes de iniciar a gravação

### Configuração do Aplicativo
1. **Instalar a versão mais recente** do Açucaradas Encomendas
2. **Configurar dados de demonstração**:
   - Adicione produtos com imagens de alta qualidade
   - Configure preços e descrições atraentes
   - Prepare um carrinho de compras com itens selecionados
3. **Verificar fluidez**:
   - Navegue por todas as telas que serão gravadas
   - Certifique-se de que não há travamentos ou lentidão

## Software de Captura

### Opções Recomendadas
- **Para Windows**: Camtasia Studio
- **Para Mac**: Screenflow

### Configurações de Captura
1. **Resolução**: Capture na resolução nativa do dispositivo
2. **Taxa de quadros**: 30 fps
3. **Qualidade**: Máxima disponível
4. **Formato de saída**: MP4 (H.264)
5. **Áudio**: Desative a captura de áudio do dispositivo

### Métodos de Captura

#### Método 1: Captura Direta via USB
1. Conecte o dispositivo ao computador via USB
2. Ative a depuração USB nas opções de desenvolvedor do dispositivo
3. Use o software de captura para espelhar a tela do dispositivo
4. Inicie a gravação no software

#### Método 2: Aplicativo de Captura no Dispositivo
1. Instale um aplicativo de gravação de tela no dispositivo (como AZ Screen Recorder)
2. Configure para gravação em 1080p a 30fps
3. Desative a gravação de áudio e o indicador de toque
4. Inicie a gravação diretamente no dispositivo

## Técnicas de Gravação

### Movimentos e Gestos
- **Movimentos suaves**: Evite movimentos bruscos ou rápidos demais
- **Velocidade consistente**: Mantenha um ritmo constante ao navegar
- **Pausas estratégicas**: Faça pequenas pausas (1-2 segundos) em elementos importantes
- **Gestos precisos**: Toque exatamente no centro dos botões e elementos interativos

### Dicas para Cada Cena

#### Cena 1: Introdução (Logo)
- Esta cena será criada na edição, não é necessário gravar

#### Cena 2: Navegação de Produtos
1. Comece na tela inicial do aplicativo
2. Navegue lentamente pela lista de produtos
3. Faça uma pausa breve em cada produto
4. Realize um scroll suave para mostrar mais produtos

#### Cena 3: Detalhes do Produto
1. Selecione o Bolo de Chocolate
2. Aguarde a tela de detalhes carregar completamente
3. Faça um scroll suave para mostrar a descrição completa
4. Destaque as opções de personalização (se disponíveis)

#### Cena 4: Carrinho de Compras
1. Adicione o produto ao carrinho
2. Navegue até a tela do carrinho
3. Mostre os produtos adicionados
4. Destaque o resumo de valores e o botão de finalização

#### Cena 5: Métodos de Pagamento
1. Clique em "Finalizar Pedido"
2. Navegue até a tela de métodos de pagamento
3. Destaque as diferentes opções disponíveis
4. Selecione uma opção (preferencialmente PIX)

#### Cena 6: Conclusão
- Esta cena será criada na edição, não é necessário gravar

## Verificação de Qualidade

### Durante a Gravação
- **Verifique cada gravação** imediatamente após concluí-la
- **Regrave imediatamente** se notar problemas como:
  - Toques imprecisos
  - Movimentos muito rápidos
  - Travamentos ou lentidão
  - Notificações ou elementos indesejados

### Após a Gravação
- **Revise todas as capturas** em tela cheia
- **Verifique a fluidez** dos movimentos
- **Confirme a legibilidade** de todos os textos
- **Verifique se todos os elementos** importantes estão visíveis

## Organização dos Arquivos

1. Crie uma pasta chamada `raw_footage` dentro de `app_preview`
2. Nomeie cada arquivo de gravação de acordo com a cena:
   - `cena2_navegacao_produtos.mp4`
   - `cena3_detalhes_produto.mp4`
   - `cena4_carrinho_compras.mp4`
   - `cena5_metodos_pagamento.mp4`
3. Mantenha os arquivos originais como backup

## Observações Finais

- Grave cada cena **várias vezes** para ter opções na edição
- Mantenha o dispositivo na **orientação retrato** durante toda a gravação
- Evite interrupções durante a gravação (coloque o telefone em modo avião)
- Certifique-se de que o aplicativo esteja na **versão final** que será publicada
- Verifique se todas as **informações exibidas** são apropriadas para publicação
