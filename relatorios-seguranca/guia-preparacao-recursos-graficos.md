## 1. Ícone do Aplicativo

### Especificações Técnicas

- **Formato**: PNG ou JPEG (sem transparência)
- **Tamanho**: 512 x 512 pixels
- **Resolução mínima**: 72 DPI
- **Tamanho máximo do arquivo**: 1 MB

### Passo a Passo para Criação

1. **Design Base**:
   - Crie um design simples e reconhecível
   - Use elementos visuais que representem confeitaria/doces
   - Mantenha consistência com a identidade visual da marca

2. **Preparação do Arquivo**:
   ```
   - Abra o Photoshop, Illustrator ou Canva
   - Crie um novo documento de 512 x 512 pixels
   - Configure a resolução para 72 DPI
   - Use o modo de cor RGB
   ```

3. **Otimização**:
   - Evite textos pequenos (serão ilegíveis em tamanhos menores)
   - Teste o ícone em diferentes tamanhos para garantir legibilidade
   - Certifique-se de que o ícone seja reconhecível mesmo em tamanho reduzido

4. **Exportação**:
   - Exporte como PNG ou JPEG
   - Verifique se o tamanho do arquivo é menor que 1 MB
   - Nomeie o arquivo como `icon_acucaradas_512.png`

### Exemplo de Design para Acucaradas Encomendas

```
Elementos recomendados para o ícone:
- Cupcake ou bolo estilizado como elemento central
- Cores: tons de rosa pastel e branco (consistentes com a marca)
- Contorno suave para destacar o elemento principal
- Fundo sólido em tom pastel para garantir contraste
```

## 2. Imagem de Destaque (Feature Graphic)

### Especificações Técnicas

- **Formato**: PNG ou JPEG (sem transparência)
- **Tamanho**: 1024 x 500 pixels
- **Resolução mínima**: 72 DPI
- **Tamanho máximo do arquivo**: 2 MB

### Passo a Passo para Criação

1. **Design Base**:
   - Crie uma imagem horizontal que capture a essência do aplicativo
   - Inclua o nome do aplicativo "Acucaradas Encomendas" em destaque
   - Use imagens de alta qualidade de produtos de confeitaria

2. **Preparação do Arquivo**:
   ```
   - Abra o Photoshop, Illustrator ou Canva
   - Crie um novo documento de 1024 x 500 pixels
   - Configure a resolução para 72 DPI
   - Use o modo de cor RGB
   ```

3. **Elementos de Design**:
   - Posicione o texto na parte central ou à esquerda
   - Use uma fonte legível e consistente com a identidade da marca
   - Adicione um slogan curto que descreva o propósito do app
   - Inclua imagens de produtos de confeitaria de alta qualidade

4. **Exportação**:
   - Exporte como PNG ou JPEG
   - Verifique se o tamanho do arquivo é menor que 2 MB
   - Nomeie o arquivo como `feature_graphic_acucaradas.png`

### Exemplo de Design para Acucaradas Encomendas

```
Composição recomendada:
- Título "Acucaradas Encomendas" em fonte elegante
- Slogan: "Doces sob medida para momentos especiais"
- Imagem de fundo: montagem com bolos, cupcakes e doces artesanais
- Paleta de cores: tons de rosa, branco e dourado
```

## 3. Screenshots

### Especificações Técnicas

- **Formato**: PNG ou JPEG (sem transparência)
- **Tamanhos recomendados**:
  - Smartphones: 16:9 (1920 x 1080 pixels)
  - Tablets: 16:10 (2560 x 1600 pixels)
- **Quantidade**: 2-8 screenshots (recomendado: 5-6)
- **Tamanho máximo por arquivo**: 2 MB

### Passo a Passo para Criação

1. **Captura de Telas**:
   - Capture as principais telas do aplicativo em dispositivos reais ou emuladores
   - Certifique-se de que o aplicativo esteja com dados de demonstração realistas
   - Capture em resolução máxima do dispositivo

2. **Telas Prioritárias para Captura**:
   ```
   - Tela inicial/catálogo de produtos
   - Detalhes de um produto (bolo/doce)
   - Processo de pedido/encomenda
   - Carrinho/finalização de compra
   - Acompanhamento de pedidos
   - Perfil do usuário/histórico de pedidos
   ```

3. **Edição e Aprimoramento**:
   - Adicione textos explicativos curtos sobre cada funcionalidade
   - Use mockups de dispositivos para enquadrar as capturas (opcional)
   - Mantenha consistência visual entre todas as screenshots

4. **Exportação**:
   - Exporte cada screenshot como PNG ou JPEG
   - Verifique se cada arquivo é menor que 2 MB
   - Nomeie os arquivos sequencialmente: `screenshot_01.png`, `screenshot_02.png`, etc.

### Exemplo de Sequência para Acucaradas Encomendas

```
Sequência recomendada:
1. Tela inicial com catálogo de produtos em destaque
2. Página de detalhes de um bolo personalizado
3. Processo de personalização de encomenda
4. Carrinho de compras com itens selecionados
5. Tela de acompanhamento de pedido
6. Histórico de pedidos/avaliações
```

## 4. Vídeo Promocional

### Especificações Técnicas

- **Formato**: MP4 ou YouTube URL
- **Resolução**: Full HD (1920 x 1080 pixels) ou superior
- **Duração**: 30 segundos a 2 minutos (ideal: 60 segundos)
- **Tamanho máximo**: 100 MB

### Passo a Passo para Criação

1. **Roteiro**:
   - Crie um roteiro simples destacando os principais benefícios do app
   - Estruture em introdução, demonstração de funcionalidades e conclusão
   - Foque nos diferenciais do Acucaradas Encomendas

2. **Gravação**:
   ```
   - Capture as interações com o app usando ferramentas de gravação de tela
   - Grave em ambiente bem iluminado se incluir pessoas ou produtos reais
   - Mantenha movimentos suaves e pausas adequadas entre as transições
   ```

3. **Edição**:
   - Adicione música de fundo suave (livre de direitos autorais)
   - Inclua legendas explicativas
   - Insira transições suaves entre as cenas
   - Adicione o logo no início e fim do vídeo

4. **Exportação**:
   - Exporte em formato MP4 com compressão H.264
   - Verifique se o tamanho do arquivo é menor que 100 MB
   - Alternativamente, faça upload para o YouTube e use a URL

### Exemplo de Estrutura para Acucaradas Encomendas

```
Roteiro básico (60 segundos):
0-10s: Introdução à marca Acucaradas e proposta de valor
10-40s: Demonstração das principais funcionalidades (navegação, personalização, pedido)
40-55s: Depoimentos/resultados (produtos entregues, clientes satisfeitos)
55-60s: Call-to-action e logo
```

## 5. Ferramentas Recomendadas

### Design Gráfico

- **Adobe Photoshop/Illustrator**: Para criação profissional de ícones e imagens
- **Canva**: Alternativa online com templates prontos
- **Figma**: Para design colaborativo

### Captura e Edição de Screenshots

- **Android Studio**: Para capturas em emuladores Android
- **Snagit**: Para capturas com edição rápida
- **Mockup Tools**: Placeit.net ou SmartMockups para enquadrar em dispositivos

### Edição de Vídeo

- **Adobe Premiere/After Effects**: Para edição profissional
- **DaVinci Resolve**: Alternativa gratuita de alta qualidade
- **CapCut**: Editor simples e eficiente para edições básicas

### Otimização de Imagens

- **TinyPNG**: Para compressão de imagens PNG sem perda de qualidade
- **ImageOptim**: Para otimização em lote de múltiplas imagens

Seguindo este guia, você garantirá que o Acucaradas Encomendas tenha uma apresentação visual profissional e atraente, aumentando suas chances de destaque e conversão.
