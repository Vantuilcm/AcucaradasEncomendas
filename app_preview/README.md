# Vídeo de Preview - Açucaradas Encomendas

## Visão Geral

Este diretório contém todos os arquivos necessários para a produção do vídeo de preview do aplicativo Açucaradas Encomendas, que será utilizado para publicação nas lojas de aplicativos (Google Play Store e App Store).

## Arquivos Disponíveis

### Documentos de Planejamento

- **[app_preview_video_script.md](./app_preview_video_script.md)**: Roteiro original com a estrutura básica do vídeo
- **[video_production_guide.md](./video_production_guide.md)**: Guia técnico original para produção do vídeo
- **[producao_video.md](./producao_video.md)**: Guia de produção detalhado com cronograma e especificações

### Documentos de Pré-Produção

- **[storyboard.md](./storyboard.md)**: Storyboard visual detalhado de cada cena do vídeo
- **[instrucoes_gravacao.md](./instrucoes_gravacao.md)**: Instruções para gravação das telas do aplicativo
- **[instrucoes_edicao.md](./instrucoes_edicao.md)**: Instruções para edição e pós-produção do vídeo
- **[roteiro_narracao.md](./roteiro_narracao.md)**: Roteiro detalhado para narração do vídeo
- **[audio_instrucoes.md](./audio_instrucoes.md)**: Instruções para música de fundo e efeitos sonoros
- **[checklist_producao.md](./checklist_producao.md)**: Checklist completo para acompanhar o progresso da produção

## Especificações Técnicas

### Vídeo

- **Duração**: 25-30 segundos
- **Resolução**:
  - Android: 1080x1920 pixels (9:16)
  - iOS: 1242x2688 pixels (opcional para futura publicação)
- **Taxa de quadros**: 30 fps
- **Formato**: MP4 (H.264)
- **Bitrate**: 8-10 Mbps

### Áudio

- **Narração**: Português brasileiro, clara e profissional
- **Música de fundo**: Suave e profissional, sem vocais
- **Formato**: AAC, 48kHz, Stereo

## Fluxo de Trabalho Recomendado

1. Revisar o **roteiro** e o **storyboard** para entender completamente o conceito
2. Preparar o dispositivo seguindo as **instruções de gravação**
3. Gravar as telas do aplicativo conforme o storyboard
4. Gravar a narração seguindo o **roteiro de narração**
5. Editar o vídeo conforme as **instruções de edição**
6. Adicionar áudio seguindo as **instruções de áudio**
7. Revisar usando o **checklist de produção**
8. Exportar nas dimensões corretas
9. Salvar o vídeo final em `store_assets/promotional_graphics/app_preview.mp4`

## Prazo de Entrega

A produção do vídeo deve ser concluída até **15/05/2025**, conforme especificado no cronograma do projeto.

## Observações Importantes

- Todos os materiais de pré-produção estão concluídos e prontos para uso
- O vídeo deve seguir rigorosamente a identidade visual da marca
- Certifique-se de que todas as informações exibidas no aplicativo são apropriadas para publicação
- Teste o vídeo final em diferentes dispositivos antes da entrega
