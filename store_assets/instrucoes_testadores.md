# Instruções para Testadores - Açucaradas Encomendas

## Introdução
Obrigado por participar do teste interno do aplicativo Açucaradas Encomendas! Sua contribuição é fundamental para identificarmos problemas e melhorarmos a experiência antes do lançamento oficial.

## Como Instalar
1. Você receberá um e-mail com um link para participar do teste
2. Clique no link e siga as instruções para instalar o aplicativo
3. Faça login com suas credenciais de teste (enviadas separadamente)

## O que Testar
Por favor, teste os seguintes fluxos principais:

### Fluxo de Compra
- Navegue pelo catálogo de produtos
- Adicione itens ao carrinho
- Finalize uma compra
- Verifique o e-mail de confirmação

### Perfil e Conta
- Crie/edite seu perfil
- Altere suas preferências
- Teste a recuperação de senha

### Rastreamento
- Verifique o status de pedidos existentes
- Teste as notificações de atualização

### Chat e Suporte
- Inicie uma conversa com o suporte
- Teste o envio de imagens no chat

## Como Reportar Problemas
Para cada problema encontrado:
1. Tire uma screenshot
2. Descreva os passos para reproduzir
3. Envie através do formulário de feedback no app (Menu > Ajuda > Reportar Problema)
