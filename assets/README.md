# Assets do Aplicativo Açucaradas Encomendas

Este diretório contém todos os recursos gráficos necessários para a publicação do aplicativo nas lojas.

## Estrutura

- `icon.png` - Ícone principal do aplicativo (1024x1024px)
- `adaptive-icon.png` - Ícone adaptativo para Android (108x108px)
- `splash.png` - Tela de splash (1242x2436px)
- `notification-icon.png` - Ícone para notificações
- `favicon.png` - Favicon para versão web

## Requisitos de Imagens

### Ícones

- Android: 512x512px (PNG)
- iOS: 1024x1024px (PNG)
- Ícone adaptativo Android: 108x108px (foreground e background)

### Splash Screen

- Resolução base: 1242x2436px (PNG)
- Proporção 16:9 para Android

### Screenshots

- Android: 1080x1920px (mínimo 2 screenshots)
- iOS: 1242x2688px (iPhone XS Max), 1125x2436px (iPhone X/XS), 1242x2208px (iPhone 8 Plus)
- Mínimo 3 screenshots por plataforma
