# Guia de Screenshots para Google Play

## Requisitos de Screenshots

### Formatos Necessários
- **Smartphone**: 1080 x 1920 pixels (16:9)
- **Tablet**: 1920 x 1200 pixels (16:10)
- **TV**: 1920 x 1080 pixels (16:9)
- **Wear OS**: 384 x 384 pixels (1:1)

### Quantidade
- Mínimo: 2 screenshots
- Máximo: 8 screenshots por tipo de dispositivo

### Formato de Arquivo
- JPEG ou PNG de 24 bits (sem transparência)
- Tamanho máximo: 8 MB por imagem

## Screenshots Recomendados

Para o app Açucaradas Encomendas, recomendamos capturar as seguintes telas:

1. **Tela Inicial/Catálogo** - Mostrando os produtos em destaque
2. **Detalhes do Produto** - Exibindo um produto específico com descrição
3. **Carrinho de Compras** - Com itens adicionados
4. **Processo de Checkout** - Mostrando opções de pagamento
5. **Rastreamento de Pedidos** - Interface de acompanhamento
6. **Perfil do Usuário** - Mostrando as informações da conta
7. **Chat de Atendimento** - Interface de comunicação
8. **Promoções/Descontos** - Exibindo ofertas especiais

## Como Capturar

### No Dispositivo Físico
1. Pressione simultaneamente os botões de energia e diminuir volume
2. As capturas são salvas na galeria do dispositivo

### No Emulador
1. Clique no botão de screenshot na barra lateral
2. Ou use o atalho Ctrl+S

## Dicas para Screenshots Eficazes

1. **Use dados realistas** - Preencha o app com conteúdo que pareça real
2. **Destaque recursos principais** - Mostre as funcionalidades mais importantes
3. **Mantenha consistência visual** - Use o mesmo tema/estilo em todas as imagens
4. **Evite texto excessivo** - As imagens devem ser autoexplicativas
5. **Considere localização** - Para mercados internacionais, use o idioma apropriado

## Próximos Passos

1. Capture os screenshots conforme as especificações acima
2. Salve-os na pasta `assets/screenshots/android` do projeto
3. Faça upload no Google Play Console durante a configuração do teste interno
