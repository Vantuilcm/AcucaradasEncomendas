### 🏢 Arquitetura Multi-tenant (SaaS)

O GrowthOS utiliza uma arquitetura de **Isolamento de Dados por Tenant**:
- Cada cliente possui seu próprio diretório em `saas/data/{tenant_id}/`.
- Histórico de releases, experimentos e decisões são completamente isolados.
- APIs dedicadas via `saas-api.js` para integração com sistemas externos.
